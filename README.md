# 🎨 Sense Shade

### *Feel Colors. Hear the World.*

Sense Shade is an accessibility-focused iOS application designed to help individuals with vision loss **experience colors through sound and speech**.
Built with a human-centered approach, the app transforms visual information into intuitive auditory feedback, enabling users to interact with colors in a completely new way.

* Winner of the Apple Swift Student Challenge 2026 🌍

---

## ✨ Features

### 🎧 Color → Sound Mapping

* Translates colors into **distinct musical tones**
* Enables users to *feel* colors through audio patterns
* Designed using intuitive mappings (brightness, intensity, tone)

### 🖼️ Tap-to-Identify Colors

* Upload an image and **tap anywhere**
* Instantly hear the color at that point
* Provides clear spoken output (e.g., *“Deep blue”*, *“Soft pastel pink”*)

### ♿ Accessibility-First Design

* Minimal and intuitive UI
* Optimized for VoiceOver and assistive usage
* Focus on independence and ease of use

---

## 🧠 How It Works

* Extracts pixel color data using image processing
* Converts color values (RGB/HSV) into audio parameters
* Maps:

  * Brightness → Pitch
  * Saturation → Intensity
  * Hue → Sound variation
* Uses speech synthesis to provide real-time color descriptions
